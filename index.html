🔒 Chiffrement des photos KYC
Les nouvelles demandes KYC sont chiffrées automatiquement. Ce bouton chiffre aussi les demandes déjà présentes en base, envoyées avant cette mise à jour. Sans effet sur celles déjà chiffrées — peut être relancé sans risque.
✓ Comptes vérifiés — noms manquants
Corrige les comptes déjà marqués "vérifié" mais dont le nom légal n'a pas été enregistré (typiquement des comptes approuvés avant l'ajout de cette fonctionnalité). Sans effet sur les comptes déjà corrects — peut être relancé sans risque.
🚩 Transactions suspectes
Signalées automatiquement selon 3 critères : trop de transactions en peu de temps, montant très supérieur aux habitudes du compte, ou premier envoi à ce destinataire avec un montant élevé. Aucune transaction n'est bloquée — c'est une alerte à vérifier, pas un blocage. Seuils modifiables dans Réglages.

Modifier le pays d'un utilisateur
Le pays est fixé à l'inscription et verrouillé côté utilisateur. À utiliser uniquement pour un vrai déménagement confirmé — chaque modification est journalisée.
⚠ Supprimer définitivement un compte
Action irréversible. Supprime le compte, le portefeuille, l'historique KYC et tout ce qui lui est associé — le numéro redevient totalement disponible pour une nouvelle inscription, comme s'il n'avait jamais existé. Réservé aux comptes de test ou erreurs d'inscription. Les transactions déjà effectuées restent visibles côté destinataire/expéditeur (jamais supprimées, pour ne pas fausser leur historique).
Trouver une transaction par numéro
